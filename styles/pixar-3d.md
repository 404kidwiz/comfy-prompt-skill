# Style snippet: Pixar 3D Animation

Append to Look line for Pixar / modern 3D animation register.

```
Style: Pixar 3D animation aesthetic.
Soft rounded geometry, exaggerated proportions on characters (large eyes, expressive features).
Warm cinematic color grading — vibrant but never oversaturated.
Subsurface scattering on skin and soft surfaces. Realistic physics with stylized geometry.
Global illumination with soft shadows, key + warm fill + cool rim lighting.
Cinematic depth of field, shallow focus on hero subject.
Surface details: subtle wear, hand-touched imperfection, hero props slightly stylized.
Composition: classical cinematic — clear hierarchy, intentional framing.
Visual reference: Toy Story, Up, Coco, Soul — Pixar feature aesthetic.
Render: high-quality offline render, no real-time game-engine look.
```

## Use cases
- Character-driven animated stills
- Children's content
- Family entertainment imagery
- Stylized concept art
- Animated film keyframes

## Compatible models
- Best: `stability-sd3`, `flux-pro` (with explicit 3D prompting)
- Good: `nano-banana`, `recraft`
- Avoid: pure "anime" register models
