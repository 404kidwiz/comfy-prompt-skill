# Style snippet: Anamorphic 1970s

Append to Look line for 1970s cinema register.

```
Style: Anamorphic widescreen 2.39:1 letterboxed.
Heavy 35mm film grain. Crushed blacks. Bleach-bypass color, low saturation.
Horizontal lens flares from any in-frame light source.
Slight chromatic aberration at frame edges.
Single-source key lighting, deep chiaroscuro shadows.
Color palette: muted ochre, faded teal, sodium-amber, cold gray-green.
Visual reference: Coppola, Scorsese, Pakula — Conversation / Mean Streets / Parallax View era.
```

## Use cases
- Psychological thrillers
- Crime / heist
- Neo-noir
- Period drama
- Slow-burn character work

## Compatible models
- Best: `flux-pro`, `flux-ultra`, `stability-ultra`
- Good: `nano-banana`, `stability-sd3`
- Video: `seedance` (handles grain + grade)

```
anamorphic widescreen 2.39:1 letterboxed, heavy 35mm film grain, crushed blacks, bleach-bypass color, low saturation, horizontal lens flares, slight chromatic aberration, single-source key with deep chiaroscuro, muted ochre and faded teal palette, 1970s cinema register
```
