# Style snippet: Studio Ghibli Painted

Append to Look line for Studio Ghibli / hand-painted animation register.

```
Style: Studio Ghibli painted matte aesthetic.
Soft watercolor brushwork, hand-painted backgrounds, slight visible canvas grain.
Warm desaturated palette — earth tones, soft greens, dusty pinks, faded blues.
Soft natural lighting, no harsh shadows, gentle rim light on subjects.
Distinctive Ghibli composition — deep negative space, atmospheric layering,
foreground silhouettes against painted distance.
Characters: rounded soft-edge linework, subtle color shading, expressive faces.
Visual reference: Miyazaki / Takahata — Spirited Away, Castle in the Sky, Mononoke era.
```
