## Recording tools

- **Terminal screenshots:** macOS Cmd+Shift+4, or `screencapture -i`
- **Animated GIFs:** [asciinema](https://asciinema.org/) → [agg](https://github.com/asciinema/agg) → GIF
- **Screen recording:** macOS Cmd+Shift+5, or Loom for narration
- **GIF compression:** `gifsicle -O3 input.gif -o output.gif` (target <5MB for LinkedIn)
- **Video compression:** `ffmpeg -i input.mov -vcodec h264 -acodec aac output.mp4` (target <50MB)

## Recording script — terminal demo

```bash
# Start asciinema
asciinema rec demo.cast --idle-time-limit 0.5 --title "comfy-prompt v3.0.0"

# Demo sequence (each line with brief pauses):
cf help | head -20
sleep 2
cf lint "epic 8k masterpiece, amazing" --model seedance --type video
sleep 3
cf compose "matte black mug" --template product --vertical viral-hook --style cyberpunk-blade-runner --print-only
sleep 4
cf jobs budget
sleep 3
cf dash
sleep 5
# Ctrl-D to stop

# Convert to GIF
agg demo.cast demo.gif --theme monokai --font-size 14 --speed 1.2
gifsicle -O3 demo.gif -o demo-optimized.gif
```

## Visual brand for the launch post

Match the hero image generated by `hero-promo.sh`:
- Deep navy + charcoal backgrounds
- Amber accent (not orange, not yellow)
- Cyan keyboard backlight cue
- Slight anamorphic letterbox
- 35mm grain register

Slide divider color: `#15151a` (matches gallery.py `--bg-card`)
Title font: bold geometric sans (Inter, Geist, or SF Pro Display)
Subtitle: monospace (JetBrains Mono, Geist Mono)
