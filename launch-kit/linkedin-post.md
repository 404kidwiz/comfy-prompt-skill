## Version A — Builder Story (Recommended)

I just shipped v3.1.0 of comfy-prompt — a Claude Code skill that turns every Comfy Cloud generation into a structured, cost-tracked, brand-aware workflow.

The problem it solves: prompt engineering for image and video models is still copy-paste from one model to another. Flux wants cinematic vocabulary. DALL-E wants natural sentences. Seedance needs explicit motion verbs. Switching models means rewriting prompts. Tracking spend means digging through CLI history. Saving brand references means a folder of unnamed PNGs.

v3.1.0 ships:

→ 20 slash subcommands (`/comfy:image`, `/comfy:video`, `/comfy:character`, `/comfy:storyboard`, `/comfy:compose`, `/comfy:variants`, `/comfy:budget`, `/comfy:dash`...)

→ A `cf` shell wrapper with `--platform tiktok|reel|wide|square` shortcuts that auto-detect each model's correct dimension flags

→ Schema introspection — `cf` reads `comfy generate schema` at runtime, catches new models automatically, applies model-specific constraints (Flux requires /32 dimensions, Pika uses camelCase float, DALL-E uses size strings — the skill handles all of it)

→ Pre-spend lint (catches missing MCSLA layers, off-list models, weak phrasing before you spend credits)

→ Cost tracker with per-model tier estimates ($0.01 nano-banana → $0.60 seedance) and monthly budget command

→ HTML output galleries auto-generated from any output dir

→ Reference image library (`cf refs add brand-x ./hero.png` → reuse in any prompt)

→ Variant generator (one prompt → 4 lighting setups, 5 framings, or 4 moods automatically)

→ Cross-model prompt translator (Flux prompt → DALL-E prompt, automatic re-phrasing)

→ Auto-poll daemon for async video jobs (no more manual `resume`)

→ Content-hash dedup (identical re-runs return existing output free)

→ 38 pytest unit tests + 9 bats integration tests (mocked, zero spend)

→ TUI dashboard, project scaffolding, EXIF embedding, --dry-run on every recipe

Built on stdlib Python + bash. No deps. MIT licensed.

**The skill promotes itself.** The poster in this post, the demo reel, the style variant pack — all generated by scripts that ship inside the skill. Same workflow you'd use to build your own:

`cf gen flux-ultra "..." --platform wide` — wide hero
`cf vid seedance "..." --platform wide` — demo reel
`cf gen nano-banana "..."` — typographic posters

Below: same skill, same `comfy-prompt` title, **4 different visual registers** (Ghibli, Film-Noir, Cyberpunk, Pixar 3D). Each one cost $0.01 via nano-banana. Total launch-kit cost: ~$1.84.
