## Version A — Builder Story (Recommended)

I just shipped v3.0.0 of comfy-prompt — a Claude Code skill that turns every Comfy Cloud generation into a structured, cost-tracked, brand-aware workflow.

The problem it solves: prompt engineering for image and video models is still copy-paste from one model to another. Flux wants cinematic vocabulary. DALL-E wants natural sentences. Seedance needs explicit motion verbs. Switching models means rewriting prompts. Tracking spend means digging through CLI history. Saving brand references means a folder of unnamed PNGs.

v3.0.0 ships:

→ 20 slash subcommands (`/comfy:image`, `/comfy:video`, `/comfy:character`, `/comfy:storyboard`, `/comfy:compose`, `/comfy:variants`, `/comfy:budget`, `/comfy:dash`...)

→ A `cf` shell wrapper with `--platform tiktok|reel|wide|square` shortcuts that auto-set aspect ratios

→ Pre-spend lint (catches missing MCSLA layers, off-list models, weak phrasing before you spend credits)

→ Cost tracker with per-model tier estimates ($0.01 nano-banana → $0.60 seedance) and monthly budget command

→ HTML output galleries auto-generated from any output dir

→ Reference image library (`cf refs add brand-nova ./hero.png` → reuse in any prompt)

→ Variant generator (one prompt → 4 lighting setups, 5 framings, or 4 moods automatically)

→ Cross-model prompt translator (Flux prompt → DALL-E prompt, automatic re-phrasing)

→ Auto-poll daemon for async video jobs (no more manual `resume`)

→ TUI dashboard, project scaffolding, EXIF embedding, content-hash dedup

Built on stdlib Python + bash. No deps. MIT licensed.

The skill promotes itself: the hero image and demo video in this post were generated by running `launch-kit/hero-promo.sh` and `launch-kit/demo-video.sh` — both ship inside the skill. Fork the repo, customize the brand.yaml, run the scripts, and you've got your own launch assets.

GitHub link in comments. Building production AI workflows? Let me know what you'd add.

## Version B — Demo-First (Tighter)

Shipped v3.0.0 of comfy-prompt today. 20 slash commands, 13 Python scripts, 8 recipes, a shell wrapper.

It's a Claude Code skill that turns Comfy Cloud (BFL Flux, Seedance, Pika, Runway, Vidu, DALL-E, Ideogram, all 30+ models) into a single coherent workflow:

```
cf gen flux-pro "cinematic portrait, rain alley" --platform wide
cf character "weathered space pilot, late 30s"
cf compose "matte black mug" --template product --vertical viral-hook --style cyberpunk-blade-runner
cf jobs budget
cf dash --watch
```

The hero image above? Generated by `launch-kit/hero-promo.sh` — a real shell script that ships with the skill. Same for the demo video. Eat your own dog food, post the dog food.

What it gives you over `comfy generate` raw:

• Pre-spend lint (catches bad prompts before they cost money)
• Cost tracking + monthly budget command
• Brand config injects palette/mood/lighting into every prompt
• Reference library you actually find again 3 months later
• HTML gallery for client preview
• Cross-model prompt translation
• 20 slash subcommands for Claude Code workflow

Stdlib Python. MIT. Fork it.
