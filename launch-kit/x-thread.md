**3/**
Solution: schema-aware workflow primitives.

```
cf gen flux-ultra "..." --platform wide
cf vid seedance "..." --platform tiktok --async
cf compose "..." --template product --vertical viral-hook --style cyberpunk
cf budget
```

The skill reads each model's schema at runtime. New models auto-detected.
