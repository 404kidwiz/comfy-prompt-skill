**2/**
The problem:

Every model wants different prompt language.
Flux: cinematic vocab.
DALL-E: natural sentences.
Seedance: explicit motion verbs over N seconds.

You end up rewriting prompts 10× per session, losing track of spend, and burying brand references in unnamed PNGs.

---

**3/**
Solution: workflow primitives.

```
cf gen flux-pro "rain-soaked detective" --platform wide
cf character "weathered pilot, late 30s"
cf storyboard "chase" "Tokyo alley" "cyberpunk"
cf budget
```

Each command runs the right preflight, organizes output, embeds metadata.

---

**4/**
Pre-spend lint catches bad prompts before they cost money:

```
$ cf lint "epic 8k masterpiece, amazing" --model seedance --type video
❌ video prompt has no explicit camera vocabulary
❌ no action verb — Action layer weak
⚠ weak phrasing: epic, 8k, masterpiece
```

Saved my last $15 of credits.
